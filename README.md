# 📍 Présentation API Geolocation

## 📝 Contexte

Ce projet a été réalisé dans le cadre d’un cours, avec pour objectif de présenter une API en **5 minutes**.

Le but était de :
- Expliquer le fonctionnement d’une API
- Montrer ses possibilités
- Faire une démonstration simple et rapide

## 🌍 API utilisée

L’API utilisée est la **Geolocation API**, qui permet de récupérer la position géographique d’un utilisateur (latitude et longitude) directement depuis son navigateur.

## 🚀 Démo

Le site propose :
- Un bouton pour obtenir la position de l’utilisateur
- L’affichage de la localisation sur une carte

## 🎯 Objectif pédagogique

Ce projet permet de comprendre :
- Comment utiliser une API côté client
- Comment gérer les permissions utilisateur
- Comment exploiter des données en temps réel
